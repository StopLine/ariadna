# ariadna
Vscode plugin for use case researching

Плагин для навигации по коду в виде древовидного набора закладок

## Цели

Основная цель -- дать наглядное представление ключевых путей исполнения в рамках заданного use case.

- быстрое погружения в контекст задачи
- передача контекста разбора use case коллегам
- расширение и увеличение глубины анализа за счёт наглядного представления и возможности фиксации наблюдений
- повышение эффективности работы для людей с ослабленными когнитивными функциями и/или дефицитом внимания

## Возможности

- задание и просматр дерева закладок к коду для иллюстрации основной линии выполнения
- добавление комментариев к закладкам, отображаемым в панели навигации
- быстрые визуальные отметки в виде эмодзи
