## Why

В панели "Node Details" редактирование полей `caption` и комментариев срабатывает по одинарному клику, что приводит к случайному открытию диалогов редактирования при навигации по дереву. Пользователи ожидают стандартного поведения VS Code TreeView — редактирование по двойному клику.

## What Changes

- Изменить триггер редактирования caption с одинарного клика на двойной клик
- Изменить триггер редактирования комментариев с одинарного клика на двойной клик
- Сохранить возможность просмотра значений полей (одинарный клик для выбора)

## Capabilities

### New Capabilities
- `double-click-edit-trigger`: Механизм обработки двойного клика для запуска редактирования полей в Node Details panel
